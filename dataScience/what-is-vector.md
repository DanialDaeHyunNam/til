## What is vector?

Vector는 물리학자, 수학자, 프로그래머는 각각 다른 개념으로 사용된다. 물리학자에게 Vector는 크기와 방향을 가진 힘을 의미하고, 프로그래머에게는 List를 의미한다. 아래에 참고 영상을 보면 직관적 이해에 도움이 된다.

Numpy 코드로 Vector A = (2, 3)은
```python
import numpy as np

np.array([[2], [3]])
//or
np.array([2, 3])
```
으로 표현한다.
